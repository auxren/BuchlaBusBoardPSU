# BuchlaBusBoardPSU
Powered bus board for Buchla synths. Takes in 15+VAC and outputs +/-15V, 12v, 5v with the edge connectors

On V1, the input and ground pins are swapped on the +5V and +12V regulator!!!! Be careful
